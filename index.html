About Cyberpunk RED Generator

A standalone interactive character manager for Cyberpunk RED by R. Talsorian Games. This tool lets you create, customise, save, and print characters with full support for stats, skills, role abilities, gear, cyberware, lifepath generation, and multiclassing.

All game data is based on the Cyberpunk RED Core Rulebook and the Black Chrome supplementary sourcebook. This is a fan-made tool intended for personal use at the table.

Features

  • 10 Stats with point-buy system (62 points, min 2 / max 8)
  • All 10 Roles with detailed rank-by-rank ability descriptions
  • Multiclass support — take a secondary role once your primary reaches Rank 4
  • Skills grouped by linked stat with search, ranks, and item bonuses
  • Weapons, Armor, Cyberware, Fashion & Gear with add/remove management
  • Custom item entry for all gear categories
  • Lifepath generation with randomise button
  • Character save/load/delete (localStorage), JSON export/import
  • Combined print layout optimised for physical character sheets
  • Dark/light theme toggle
