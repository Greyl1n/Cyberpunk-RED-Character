About Cyberpunk RED Generator

A fully self-contained, highly-optimised interactive character manager for Cyberpunk RED by R. Talsorian Games. Create, customise, save, and print characters with full support for stats, skills, role abilities, gear, cyberware, lifepath generation, and multiclassing. No server, no build tools, no internet — just open the file.

The v3.3 Update completely overhauls the Lifepath tab with standard static dropdowns and dynamic lists for Friends, Enemies, and Lovers. It also brings the SP and Armor Penalty values for all armor directly in line with the Cyberpunk RED Core Rulebook.

The v3.2 Update introduces recursive Cyberware options (such as the Modular Finger Cyberhand and its cyberfingers) with accurate nested Humanity Cost calculations, and resolves various point calculation bugs during character creation.

Features

  • 10 Stats with point-buy system (62 points, min 2 / max 8)
  • All 10 Roles with detailed rank-by-rank ability descriptions
  • Multiclass support — take a secondary role once your primary reaches Rank 4
  • 86 Skills grouped by linked stat with search, ranks, and item bonuses
  • 140+ Weapons mapped accurately to their Cyberpunk RED skills (e.g. Shoulder Arms, Handgun, Heavy Weapons)
  • Add/remove management with cost deduction for Armor, Cyberware, Fashion & Gear
  • Ammunition Consumption — track remaining rounds directly on your sheet
  • Custom item entry for all gear categories
  • Categorised item selectors grouped logically
  • Lifepath generation with randomise button
  • Character save/load/delete (localStorage), JSON export/import
  • Combined print layout optimised for physical character sheets
  • Random character generator — one-click complete, playable character
  • Netrunner dark theme with native CSS variables
  • Random Humanity Cost (HC) rolls and custom inputs for Cyberware
  • Optimised mobile styling with seamless skill adjustments
  • Polished print-friendly layout with customisable background dimming
  • Mobile-responsive layout with touch-optimised controls
  • Single-file or Modular modes — zero external dependencies, works offline
