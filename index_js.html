About Cyberpunk RED Generator

A fully self-contained, mobile-friendly interactive character manager for Cyberpunk RED by R. Talsorian Games. This tool lets you create, customise, save, and print characters with full support for stats, skills, role abilities, gear, cyberware, lifepath generation, and multiclassing. No server, no build tools, no internet — just open the file.

All game data is based on the Cyberpunk RED Core Rulebook, Black Chrome, and numerous official DLC releases. This is a fan-made tool intended for personal use at the table.

Features

  • 10 Stats with point-buy system (62 points, min 2 / max 8)
  • All 10 Roles with detailed rank-by-rank ability descriptions
  • Multiclass support — take a secondary role once your primary reaches Rank 4
  • 86 Skills grouped by linked stat with search, ranks, and item bonuses
  • 140+ Weapons, Armor, Cyberware, Fashion & Gear with add/remove management
  • Custom item entry for all gear categories
  • Categorised item selectors grouped by weapon type, gear category, or cyberware type
  • Lifepath generation with randomise button
  • Character save/load/delete (localStorage), JSON export/import
  • Combined print layout optimised for physical character sheets
  • Random character generator — one-click complete, playable character
  • Dark/light theme toggle
  • Mobile-responsive layout with touch-optimised controls
  • Single-file app — zero external dependencies, works offline
