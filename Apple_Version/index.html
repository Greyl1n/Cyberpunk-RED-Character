About Cyberpunk RED Generator (Playtest Build)

A fully self-contained, highly-optimised interactive character manager for Cyberpunk RED by R. Talsorian Games. Create, customise, save, and print characters with full support for stats, skills, role abilities, gear, cyberware, lifepath generation, and multiclassing. No server, no build tools, no internet - just open the single monolith HTML file in any browser.

Welcome to the Playtest Build! This version introduces massive updates across the board, moving beyond a simple character sheet into a full character management suite. Key updates include deeply interactive Role Abilities (Exec Team Members, Nomad Vehicle tracking, Lawman Backup, Medtech Pharmaceuticals, Netrunner Interface, and more), fully integrated Cyberware humanity and slot tracking, overhauled generic and Role-specific Lifepaths, Ammunition tracking, and seamless local state management.

Features

  • 10 Stats with point-buy system (62 points, min 2 / max 8)
  • All 10 Roles with detailed rank-by-rank ability descriptions and deeply integrated mechanical trackers
  • Interactive Role Abilities: Create and manage Exec Team Members, track Nomad Family Vehicles & Upgrades, handle Lawman Backup stats, manage Netrunner Interface abilities, and track Medtech Pharmaceuticals
  • Multiclass support - take a secondary role once your primary reaches Rank 4
  • 86 Skills grouped by linked stat with search, ranks, and item bonuses
  • 140+ Weapons mapped accurately to their Cyberpunk RED skills (e.g. Shoulder Arms, Handgun, Heavy Weapons)
  • Add/remove management with cost deduction for Armor, Cyberware, Fashion & Gear
  • Ammunition Consumption - track remaining rounds directly on your sheet
  • Dynamic Cyberware Tracking: automatically handles humanity loss, maximum humanity reduction, and custom slot constraints (e.g. 7 slots for a Cyberarm)
  • Custom item entry for all gear categories
  • Categorised item selectors grouped logically
  • Full Lifepath generation (Generic and Role-specific) with randomise button
  • Character save/load/delete (localStorage), JSON export/import
  • Combined print layout optimised for physical character sheets
  • Random character generator — one-click complete, playable character
  • Netrunner dark theme with native CSS variables
  • Random Humanity Cost (HC) rolls and custom inputs for Cyberware
  • Optimised mobile styling with seamless skill adjustments
  • Polished print-friendly layout with customisable background dimming
  • Mobile-responsive layout with touch-optimised controls
